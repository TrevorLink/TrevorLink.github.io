JavaScript|...扩展运算符

问题引入

最近在项目代码中经常会看到 ... 的操作符，不管是在 React 还是 Vue 的写法中都很通用

快速总结： ... 是 JS 中的扩展操作符，可以对变量操作，可以用在函数签名中，也可以用于组件props传递

  • 对变量操作，可以将数组/对象变量进行展开
  • 用在函数签名中，可以用于将函数签名中所有的参数都收集到指定的数组里（类似JDK8提供的可变形参）
  • 用在组件props传递，可以定制更通用化的组件
